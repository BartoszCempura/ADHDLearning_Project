Aplikacja umożliwia:
- deklarację zadań do wykonania
- utworzenie listy, zapis i wczytanie jej stanu
- określenie okna pracy dla jednego zadania oraz czasu przerwy
- na podstawie ilości zadań obliczany jest całkowityc czas wyznaczony na pracę

Aplikacja implementuje kilka z rozwiązań mających umożliwić przełamanie problemu prokrastynacji:
1. Wyznaczanie małych , prostych zadań do wykonania w krótkim okresie czasu
2. Wyznaczenie okresu pracy przeplatanego odpoczynkiem - metoda pomodoro
3. Przy uruchomieniu odliczania można wybrać ulubione nagranie które umożliwi skupienie na zadaniu np. brown noise, white noise
4. Na środku ekranu pjawia się okno odliczania czasu przerwy utrudniające pozostanie przed komputerem i przeglądanie internetu


The application allows you to:
- declare tasks to be completed
- create a list of tasks, save, and load it
- define a work window for a single task and a break time
- calculate the total time allocated for work based on the number of tasks

The application implements several solutions designed to help overcome procrastination:
1. Designate small, simple tasks to be completed in a short period of time
2. Designate work periods interspersed with rest periods - the Pomodoro method
3. When starting the countdown, you can choose your favorite recording to help you focus on the task, e.g., brown noise or white noise
4. A break time countdown window appears in the center of the screen, making it difficult to stay in front of the computer and browse the internet
